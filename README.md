# nested-polygons

Tool for creating cool visualizations of rotating, nested polygons.

## Requirements
* matplotlib
* numpy
* ffmpeg

## Usage
```python
python nested-polygons.py --help
```

## Example

```bash
python nested-polygons.py 10 decagon.mp4 --fps 20 --colors "royalblue" "silver"
```
